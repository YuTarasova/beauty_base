## Подготовка к запуску
1. Изменение .env файла:
- Вставьте пароль от MySQL Server напротив "DB_PASSWORD=", например, `DB_PASSWORD=password`.
- При необходимости отредактируйте прочие переменные.
2. Установите зависимости (пакеты перечислены в requirements.txt).
3. Инициализируйте базу данных. Для этого необходимо запустить database.py, а после завершения закомментировать последнюю строку.
## Запуск
Запустите run.py
